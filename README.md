# Calculadora de partidas Rankeadas

Este projeto foi desenvolvido como parte de um desafio prático para consolidar fundamentos de programação. O objetivo é calcular o saldo de vitórias de um herói e determinar seu nível dentro de um sistema de ranking.

🚀 Tecnologias Utilizadas
JavaScript: Linguagem principal utilizada para a lógica.

Node.js: Ambiente de execução para testar o script via terminal.

🧠 Lógica Aplicada
O código utiliza uma função que recebe o número de vitórias e derrotas, realiza o cálculo matemático e utiliza uma estrutura de decisão (if/else if) para classificar o herói nos seguintes níveis:

Vitórias,Nível
Menor que 10,Ferro
Entre 11 e 20,Bronze
Entre 21 e 50,Prata
Entre 51 e 80,Ouro
Entre 81 e 90,Diamante
Entre 91 e 100,Lendário
Maior ou igual a 101,Imortal

🛠️ Como executar
Certifique-se de ter o Node.js instalado.

Clone o repositório ou copie o código do arquivo .js.

No terminal, execute o comando: node index.js
